How It Works

  1. Upload - Upload your CSV file with beneficiary addresses
  2. Review - Review data and set depot address
  3. Geocode - Convert addresses to coordinates
  4. Generate - Create optimized routes (max 4 stops each)
  5. Download - Get GPX files for OsmAnd navigation

Expected CSV Format

Column Required Description
name Yes Beneficiary's full name
address Yes Full delivery address (street, city, zip)
phone No Contact phone number
household_size No Number of people
items_needed No Items to deliver
special_items No Special items (diapers, formula, etc.)
notes No Delivery notes
