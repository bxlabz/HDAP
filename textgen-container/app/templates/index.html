How It Works

  1. With Route Manifest: Upload both the CSV and the route_manifest.json from the routing app. Packing slips will be numbered to match delivery routes (Route 1 Stop 1, Route 1 Stop 2, etc.)
  2. Without Manifest: Upload just the CSV. Packing slips will be numbered sequentially (1, 2, 3...)

Expected CSV Format

Your CSV file should include the following columns:

Column Required Description
name Yes Beneficiary's full name
phone Yes* Phone number (any format)
address Yes Delivery address
household_size Yes* Number of people in household
items_needed Yes* List of items to deliver
special_items No Diapers, formula, pet food, etc.
contact_preference No How to contact beneficiary
notes No Additional notes

* Items marked Yes* will generate warnings if missing but won't prevent processing.
